Finding the Product Variant ID

To locate a Product Variant ID:

  1. Go to Shopify Admin > Products.
  2. Select the product you want.
  3. Click the specific variant.

The Product Variant ID will appear in the URL. For example, for the Inventory-Only ODG Kydex variant, the URL is:

https://admin.shopify.com/store/tenicor/products/7307312627787/variants/41320119304267

The Product Variant ID is at the end of the URL. Here, the Product Variant ID is:

41320119304267

Adding the Variant ID to the JSON

Insert this ID into the JSON text field below using the following format:

{
  "ABC": "gid://shopify/ProductVariant/1234567890",
  "ODG": "gid://shopify/ProductVariant/41320119304267"
}

Now, any product variant whose SKU contains "ODG" will have its inventory quantity automatically synced to match the inventory of Inventory-Only ODG Kydex.
